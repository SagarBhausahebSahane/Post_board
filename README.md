## Features Postboard — React CRUD App
- Create posts with Title, Description, and Image (PNG/JPG/JPEG)
- View all posts in a responsive card grid
- Edit posts via pre-filled modal
- Delete posts with confirmation
- All data persisted in localStorage
- Live search/filter
- Fully responsive on all screen sizes
- Custom CSS only (no external UI libs except React)

## Run Locally

### Prerequisites
- Node.js v16 or higher
- npm

### Steps

```bash
# 1. Navigate into the project folder
cd postboard

# 2. Install dependencies
npm install

# 3. Start the development server
npm start
```
The app will open at **http://localhost:3000**
